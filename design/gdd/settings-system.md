# 设置系统 (Settings System)

> **Status**: Designed
> **Author**: User + Claude Code
> **Last Updated**: 2026-03-30
> **Implements Pillar**: 碎片友好

## Overview

**设置系统**负责管理玩家偏好设置，包括音量控制、通知设置、显示设置等，并提供设置UI界面。

系统提供设置的读取、修改、保存、重置功能。所有设置持久化到存档系统，游戏启动时自动加载。

**为什么需要这个系统**: 没有设置系统，玩家无法个性化游戏体验。音量调节、通知开关等基础功能缺失，降低玩家满意度。系统化的设置管理让玩家自由定制游戏，提升体验。

## Player Fantasy

玩家期望体验：

**1. 个性化控制**
- 随时调节音量，适合不同环境（安静/嘈杂）
- 开关通知，避免打扰
- 自定义游戏体验

**2. 设置持久化**
- 设置自动保存，下次启动保持
- 跨设备同步（后期云存档）
- 一键重置，恢复默认

**3. 简洁易用**
- 设置分类清晰，快速找到
- 滑块/开关操作直观
- 说明文案简洁明了

## Formulas

### 设置验证
```
validate_setting(key, value) =
    CASE key:
        "volume_master", "volume_music", "volume_sfx":
            RETURN clamp(value, 0.0, 1.0)
        "auto_save_interval":
            RETURN clamp(value, 10, 300)
        "mute_all", "notifications_enabled", etc.:
            RETURN value AS bool
        ELSE:
            RETURN error("未知设置项")
```

### 设置变更通知
```
on_setting_changed(key, value):
    1. 验证设置值
    2. 更新 settings[key]
    3. 触发相关系统回调
    4. 自动保存到存档
```

**示例**:
- 音量变更 → 通知音频系统更新音量
- 通知开关变更 → 更新通知系统状态
- 自动保存间隔变更 → 更新存档系统定时器

### 设置重置
```
reset_to_defaults():
    FOR each (key, default_value) IN DEFAULT_SETTINGS:
        settings[key] = default_value
    save_settings()
    notify_all_systems()
```

**默认值来源**: `DEFAULT_SETTINGS` 常量字典

## Edge Cases

| 边缘情况 | 处理方式 | 影响范围 |
|----------|----------|----------|
| **存档中无设置数据** | 使用默认值，初始化设置 | 游戏启动 |
| **设置值超出范围** | Clamp到有效范围，记录警告 | 设置加载 |
| **设置键不存在** | 使用默认值，记录警告 | 设置加载 |
| **设置类型错误** | 转换为正确类型，失败则使用默认值 | 设置加载 |
| **快速连续修改设置** | 即时生效，合并保存请求 | 设置UI |
| **重置设置时游戏运行中** | 即时生效，通知所有相关系统 | 设置重置 |
| **云同步设置冲突** | 本地优先，显示冲突提示 | 云同步 |
| **设置保存失败** | 重试3次，失败则显示错误提示 | 设置保存 |
| **设置UI打开时切换场景** | 保存当前设置，关闭UI | 场景切换 |
| **多设备同时修改设置** | 后修改的覆盖（云同步场景） | 云同步 |

## Tuning Knobs

| Knob | 类型 | 默认值 | 范围 | 影响范围 |
|------|------|--------|------|----------|
| `default_volume_master` | float | 1.0 | 0.0-1.0 | 新玩家默认主音量 |
| `default_volume_music` | float | 0.8 | 0.0-1.0 | 新玩家默认音乐音量 |
| `default_volume_sfx` | float | 1.0 | 0.0-1.0 | 新玩家默认音效音量 |
| `default_auto_save_interval` | int | 30 | 10-300 | 自动保存间隔（秒） |
| `volume_step` | float | 0.05 | 0.01-0.1 | 音量滑块步进值 |
| `settings_save_debounce` | float | 0.5 | 0.1-2.0 | 设置保存防抖时长（秒） |

### 调参指南

**`default_auto_save_interval`（自动保存间隔）**
- **调低**: 更频繁保存，更安全但可能影响性能
- **调高**: 更少保存，性能更好但可能丢失更多进度
- **观察指标**: 玩家退出后进度丢失率

**`volume_step`（音量步进）**
- **调低**: 滑块更精细，但操作更慢
- **调高**: 滑块更粗粒度，但操作更快
- **观察指标**: 玩家调整音量的次数和满意度

**`settings_save_debounce`（保存防抖）**
- **调低**: 设置更快保存，但可能频繁IO
- **调高**: 设置延迟保存，减少IO但可能丢失
- **观察指标**: 设置保存频率和IO性能
