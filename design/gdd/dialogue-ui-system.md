# 对话UI系统 (Dialogue UI System)

> **Status**: Designed
> **Author**: User + Claude Code
> **Last Updated**: 2026-03-30
> **Implements Pillar**: 沉浸修仙

## Overview

**对话UI系统**负责游戏中的对话界面，主要用于BOSS面试场景，展示AI面试官的对话内容、玩家选择、回答输入。

**为什么需要这个系统**: BOSS面试是核心玩法，需要专业的对话界面支持文字输入、选项选择、AI回复展示。

## Player Fantasy

玩家期望体验：

**1. 沉浸的对话体验**
- 像与真人面试官对话
- 对话框清晰、文字易读
- 打字机效果增强代入感

**2. 流畅的交互**
- 快速输入答案
- 选项点击响应灵敏
- 支持键盘快捷键

**3. 专业的面试感**
- 布局像真实面试场景
- BOSS形象+名称显示
- 对话历史可回顾

## Detailed Design

### Core Rules

#### 对话界面结构

```
┌─────────────────────────────────────────────────────────────┐
│  ┌─ BOSS信息栏 ───────────────────────────────────────┐   │
│  │ [头像] 大厂长老 - Java架构师                       │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ┌─ 对话区域 ─────────────────────────────────────────┐   │
│  │                                                     │   │
│  │  长老: "年轻人，吾且问你..."                       │   │
│  │                                                     │   │
│  │  你: "HashMap底层是..."                            │   │
│  │                                                     │   │
│  │  长老: "不错，那继续追问..."                       │   │
│  │                                                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ┌─ 输入区域 ─────────────────────────────────────────┐   │
│  │ ┌─────────────────────────────────────────────┐   │   │
│  │ │ [输入答案...]                                │   │   │
│  │ └─────────────────────────────────────────────┘   │   │
│  │                          [提交答案]               │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 对话类型

| 类型 | 说明 | UI形式 |
|------|------|--------|
| **BOSS台词** | AI生成的对话 | 文字+打字机效果 |
| **玩家选择** | 预设选项 | 选项按钮列表 |
| **玩家输入** | 自由文字/代码 | 输入框 |
| **系统提示** | 面试状态提示 | 气泡提示 |

### States and Transitions

```
┌─────────────────┐
│   等待BOSS说话   │
└────────┬────────┘
         │ BOSS台词显示
         ↓
┌─────────────────┐
│   玩家输入/选择   │
└────────┬────────┘
         │ 提交
         ↓
┌─────────────────┐
│   AI处理中       │
└────────┬────────┘
         │ AI回复
         ↓
┌─────────────────┐
│   显示AI回复     │
└────────┬────────┘
```

### Interactions with Other Systems

| 系统 | 数据流向 | 接口 |
|------|----------|------|
| **场景管理系统** | 读取 | 加载对话场景 |
| **BOSS面试系统** | 双向 | 接收对话数据、返回玩家输入 |

## Formulas

### 打字机速度
```
typewriter_speed = 30  // 字符/秒
display_duration = text_length / typewriter_speed
```

### 对话历史限制
```
max_history_lines = 50
IF history.size() > max_history_lines:
    remove_oldest_lines()
```

## Edge Cases

| 边缘情况 | 处理方式 |
|----------|----------|
| **输入为空** | 提示"请输入答案" |
| **输入过长** | 限制最大字符数，超出截断 |
| **AI回复慢** | 显示"思考中..."动画 |
