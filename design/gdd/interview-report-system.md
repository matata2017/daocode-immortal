# 面试报告系统 (Interview Report System)

> **Status**: Designed
> **Author**: User + Claude Code
> **Last Updated**: 2026-03-30
> **Implements Pillar**: 实用至上, 成长可见

## Overview

**面试报告系统**负责生成和展示BOSS面试后的详细报告，包括答题统计、知识点分析、AI反馈总结、改进建议。

**为什么需要这个系统**: 面试报告让玩家知道自己哪里做得好、哪里需要改进，是学习闭环的重要环节。

## Player Fantasy

玩家期望体验：

**1. 清晰的自我认知**
- 知道自己答对/答错了哪些
- 知道哪些知识点薄弱
- 知道如何改进

**2. 可回顾的学习资料**
- 报告可以保存回顾
- AI反馈有价值，可以反复学习
- 历史报告对比进步

**3. 分享的成就感**
- 可以分享面试结果
- 展示自己的学习进度

## Detailed Design

### Core Rules

#### 报告内容

| 模块 | 内容 |
|------|------|
| **面试概览** | BOSS名称、面试时间、总题数、正确率 |
| **答题详情** | 每道题的对错、玩家答案、正确答案 |
| **知识点分析** | 各领域正确率、薄弱项标记 |
| **AI总结** | AI生成的面试评价和建议 |
| **修为变化** | 本次面试获得的修为 |

### Interactions with Other Systems

| 系统 | 数据流向 | 接口 |
|------|----------|------|
| **回答分析系统** | 读取 | 获取答题分析数据 |
| **存档系统** | 写入 | `save_interview_report()` |

## Edge Cases

| 边缘情况 | 处理方式 |
|----------|----------|
| **面试中断** | 生成部分报告 |
| **全对/全错** | 特殊评价文案 |
