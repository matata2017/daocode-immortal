# 代码编辑器系统 (Code Editor System)

> **Status**: Designed
> **Author**: User + Claude Code
> **Last Updated**: 2026-03-30
> **Implements Pillar**: 实用至上

## Overview

**代码编辑器系统**负责代码题的代码编辑功能，包括语法高亮、代码补全、运行测试（后期）、代码格式化。

**为什么需要这个系统**: 代码题需要实际的代码编辑体验，纯文本框无法满足代码题的需求。

## Player Fantasy

玩家期望体验：

**1. 专业的代码编写感**
- 语法高亮，清晰易读
- 代码补全，提高效率
- 缩进/格式化，代码整洁

**2. 即时的反馈感**
- 运行测试立即看到结果（后期功能）
- 错误提示清晰定位
- 提示帮助理解问题

**3. 沉浸的编程氛围**
- 暗色主题，专注编码
- 修仙风格的UI元素
- 与游戏整体风格一致

## Detailed Design

### Core Rules
#### 编辑器功能
| 功能 | 描述 |
|------|------|
| **语法高亮** | Java/GDScript 语法着色 |
| **行号显示** | 左侧行号 |
| **代码补全** | 基于上下文的智能补全 |
| **自动缩进** | 智能缩进 |
| **括号匹配** | 自动高亮匹配括号 |
| **错误提示** | 红色波浪线标记错误 |
| **字符计数** | 显示已输入字符数 |

#### 编辑器配置
```gdscript
var editor_config = {
    font_family: "Fira Code",
    font_size: 14,
    tab_size: 4,
    theme: "dark",
    line_numbers: true,
    auto_indent: true,
    max_length: 1000
}
```

### States and Transitions
```
┌─────────────────┐
│   空闲状态       │  等待输入
└────────┬────────┘
         │ 用户输入
         ↓
┌─────────────────┐
│   编辑中         │  实时高亮
└────────┬────────┘
         │ 提交/取消
         ↓
┌─────────────────┐
│   提交中         │  验证代码
└─────────────────┘
```
### Interactions
| 系统 | 数据流向 | 接口 |
|------|----------|------|
| **题库数据系统** | 读取 | 获取代码题内容 |

## Formulas
### 代码长度计算
```
code_length = input_text.length
IF code_length > max_length:
    show_warning("代码过长，可能影响评分")
```
## Edge Cases
| 边缘情况 | 处理方式 |
|----------|----------|
| **代码为空** | 提示"请输入代码" |
| **代码过长** | 显示字符数警告 |
| **语法错误** | 红色波浪线标记 |
