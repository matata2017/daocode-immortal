## 1. 设计原则

### 1.1 现代简约仙侠定义

**核心理念**：保留仙侠精神内核，但以现代设计语言呈现。

| 设计维度 | 现代简约仙侠 | 传统仙侠（避免） |
|----------|--------------|------------------|
| **控件风格** | 扁平化、圆角、简洁边框 | 写实材质、复杂纹理 |
| **边框装饰** | 简化仙侠纹样（单色线条） | 繁复云纹、雕花边框 |
| **按钮设计** | 现代扁平 + 微妙阴影 | 拟物化玉石、木牌 |
| **卡片背景** | 半透明毛玻璃 + 淡雅纹理 | 宣纸、卷轴材质 |
| **图标风格** | 线性图标 + 仙侠元素 | 写实水墨、复杂图案 |

**设计边界**：
- **保留**：仙侠配色（青、金、玄色）、仙侠术语、意境氛围
- **简化**：减少装饰细节，保持视觉清爽
- **禁止**：过度拟物、繁复纹样、影响可读性的装饰

**参考对标**：
- 《原神》UI（现代扁平 + 东方元素）
- 《崩坏：星穹铁道》UI（简洁线条 + 科幻仙侠融合）

### 1.2 温暖生机色彩理念

**情感目标**：传达修炼成长的积极氛围，鼓励型语调。

| 情感维度 | 色彩策略 | 避免方向 |
|----------|----------|----------|
| **成长** | 暖金、淡橙为主色调 | 冷色调主导 |
| **生机** | 柔和青绿点缀 | 枯黄、暗灰 |
| **希望** | 明亮但不刺眼 | 过于鲜艳或暗沉 |
| **鼓励** | 温暖渐变、柔和过渡 | 强烈对比、冷峻 |

**色彩心理学应用**：
- **暖金色**：成就、价值、进步（修为增长）
- **淡橙色**：温暖、活力、友好（日常修炼）
- **柔和青绿**：生命、成长、自然（境界提升）
- **玄色（深灰）**：稳重、深度、底蕴（背景层次）

### 1.3 标识区分境界系统

**设计策略**：界面保持统一基础色调，境界通过独立视觉元素区分。

| 区分维度 | 实现方式 | 应用场景 |
|----------|----------|----------|
| **图标** | 每个境界专属图标 | 主菜单、状态栏 |
| **徽章** | 境界徽章边框 + 底色 | 玩家信息卡 |
| **特效** | 突破时专属粒子/光效 | 突破场景 |
| **立绘** | 角色外观渐进变化 | 修炼场景、对话 |
| **文案** | 境界专属称号和描述 | 所有界面 |

**不变元素**（所有境界统一）：
- 主界面布局
- 主色调（暖金/淡橙）
- 按钮样式
- 底部导航栏
- 卡片样式

**变化元素**（随境界变化）：
- 境界图标
- 境界徽章背景色
- 突破特效颜色
- 角色立绘
- 境界称号

### 2.4 功能色（语义色）

#### 反馈状态色

| 状态 | 默认 HEX | 色盲模式 HEX | 图标 | 用途 |
|------|----------|--------------|------|------|
| **正确/成功** | `#4CAF50` | `#2196F3` | ✓ 对勾 | 答对、突破成功 |
| **错误/失败** | `#F44336` | `#FF9800` | ✗ 叉号 | 答错、突破失败 |
| **警告/中危** | `#FF9800` | `#FFC107` | ▲ 三角 | 心魔中危、注意 |
| **信息/提示** | `#2196F3` | `#2196F3` | ○ 圆点 | 一般提示信息 |

**色盲安全设计**：
- 所有状态色必须配合图标使用（双重编码）
- 红绿模式：正确=蓝色，错误=橙色
- 蓝黄模式：保持默认，增强图标提示

### 2.5 背景色层级

采用 4 层深度系统，通过明度差异建立视觉层次。

| 层级 | 名称 | HEX | RGB | 用途 |
|------|------|-----|-----|------|
| **L0** | 深层背景 | `#1A1C22` | (26, 28, 34) | 全屏背景、弹窗遮罩 |
| **L1** | 基础背景 | `#2A2D36` | (42, 45, 54) | 主界面背景 |
| **L2** | 卡片背景 | `#3A3D46` | (58, 61, 70) | 卡片、面板、列表项 |
| **L3** | 悬浮背景 | `#4A4D56` | (74, 77, 86) | 悬浮层、下拉菜单 |

**背景层级应用示例**：
```
L0: 全屏背景
  └─ L1: 主界面容器
       ├─ L2: 境界信息卡
       │    └─ L3: 修为进度条背景
       └─ L2: 快捷入口面板
```

## 3. 字体系统

### 3.1 字体选择

#### 中文字体

| 用途 | 字体名称 | 备选字体 | 特点 |
|------|----------|----------|------|
| **标题/显示** | 思源黑体 Bold | Noto Sans SC Bold | 现代、清晰、有力量感 |
| **正文/界面** | 思源黑体 Regular | Noto Sans SC Regular | 易读、通用、开源免费 |

#### 英文/数字字体

| 用途 | 字体名称 | 备选字体 | 特点 |
|------|----------|----------|------|
| **标题/显示** | Roboto Bold | Noto Sans Bold | 现代、清晰 |
| **正文/界面** | Roboto Regular | Noto Sans Regular | 易读、通用 |
| **数字/代码** | JetBrains Mono | Fira Code, Source Code Pro | 等宽、清晰 |

### 3.2 字号阶梯

采用 8px 基础单位的倍数系统。

| 级别 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| **H1** | 28px | 36px | Bold (700) | 屏幕标题 |
| **H2** | 24px | 32px | Bold (700) | 卡片标题 |
| **H3** | 20px | 28px | SemiBold (600) | 小标题、标签 |
| **Body Large** | 18px | 26px | Regular (400) | 主要正文、选项文字 |
| **Body** | 16px | 24px | Regular (400) | 次要正文、说明文字 |
| **Caption** | 14px | 20px | Regular (400) | 辅助说明、提示文字 |
| **Small** | 12px | 16px | Regular (400) | 时间戳、次要标签 |

**字号调整策略**（设置中可选）：
- **小**：所有字号 -2px
- **中**（默认）：标准字号
- **大**：所有字号 +4px
- **最小正文**：不低于 14px

### 3.3 代码字体规格（CodeEdit 场景）

| 属性 | 值 | 说明 |
|------|-----|------|
| **字体** | JetBrains Mono | 等宽字体，支持连字 |
| **字号** | 16px | 可在设置中调整 (14-20px) |
| **行高** | 24px | 1.5 倍行高 |
| **Tab 宽度** | 4 空格 | 可配置 |
| **语法高亮** | 支持关键字、字符串、注释着色 | 基础级别 |

### 3.4 字间距与行间距

| 元素类型 | 字间距 | 行间距 | 说明 |
|----------|--------|--------|------|
| **标题** | 0.02em | 1.3 | 紧凑，视觉冲击 |
| **正文** | 0 | 1.5 | 舒适阅读 |
| **按钮** | 0.01em | 1.2 | 紧凑，清晰 |
| **代码** | 0 | 1.5 | 等宽，对齐 |

## 8. 动画规范

### 8.1 过渡时长

| 类型 | 时长 | 用途 |
|------|------|------|
| **快速** | 150ms | 按钮悬停、图标切换、焦点变化 |
| **标准** | 300ms | 弹窗出现、页面切换、卡片展开 |
| **慢速** | 500ms | 突破动画、成就展示、重要反馈 |
| **特慢** | 1000ms+ | 飞升动画、开场动画 |

### 8.2 缓动函数

| 缓动名称 | CSS 值 | 用途 |
|----------|--------|------|
| **Ease Out** | `cubic-bezier(0, 0, 0.2, 1)` | 元素进入、展开 |
| **Ease In Out** | `cubic-bezier(0.4, 0, 0.2, 1)` | 状态变化、过渡 |
| **Spring** | `cubic-bezier(0.175, 0.885, 0.32, 1.275)` | 弹跳效果、活泼反馈 |
| **Linear** | `linear` | 进度条、连续动画 |

### 8.3 关键动画规格

#### 弹窗出现
- **时长**：300ms
- **缓动**：Ease Out
- **效果**：从中心放大 + 淡入
- **初始状态**：scale(0.9), opacity(0)
- **结束状态**：scale(1.0), opacity(1)

#### 答题反馈
- **正确**：绿色边框闪烁 + 对勾图标放大（500ms）
- **错误**：红色边框 + 抖动效果（300ms）

#### 修为增加
- **时长**：800ms
- **效果**：数字跳动 + 金色粒子上升
- **飘字**："+20" 从修为数字处向上飘起并淡出

#### 境界突破
- **时长**：2000ms
- **效果**：
  1. 屏幕渐暗（300ms）
  2. 粒子汇聚（800ms）
  3. 光芒绽放（500ms）
  4. 新境界徽章展示（400ms）

#### 心魔斩除
- **时长**：600ms
- **效果**：心魔图标分裂 + 消散粒子

### 8.4 可跳过性要求

| 动画类型 | 可跳过 | 跳过方式 |
|----------|--------|----------|
| **答题反馈** | 是 | 点击任意处 |
| **突破动画** | 是 | 点击"跳过"按钮 |
| **飞升动画** | 是 | 点击"跳过"按钮 |
| **页面过渡** | 否 | - |
| **弹窗动画** | 否 | - |

### 8.5 减弱动画模式

**触发条件**：设置中开启"减弱动画"或系统偏好设置

**效果**：
- 所有动画时长缩短为 0ms（瞬间完成）
- 粒子效果禁用
- 过渡效果改为瞬间切换
- 保留必要的视觉反馈（颜色变化、图标切换）

## 10. 无障碍视觉

### 10.1 焦点指示器样式

**WCAG 2.1 AA 要求**：对比度 >= 3:1

| 属性 | 值 |
|------|-----|
| **颜色** | `#FFFFFF` 白色 |
| **边框宽度** | 3px |
| **边框样式** | 实线 |
| **偏移** | 2px（与元素边缘的距离） |
| **圆角** | 与元素圆角一致 |
| **动画** | 无（避免闪烁） |

**焦点指示器应用规则**：
- 所有可交互元素必须有焦点指示器
- 焦点指示器必须清晰可见，不与背景混淆
- 焦点顺序符合逻辑（从左到右，从上到下）

### 10.2 高对比度模式配色

**触发条件**：设置中开启"高对比度模式"

| 元素 | 标准模式 | 高对比度模式 |
|------|----------|--------------|
| **背景** | `#2A2D36` | `#000000` 纯黑 |
| **文字** | `#FAF7F2` | `#FFFFFF` 纯白 |
| **次要文字** | `#B0B3BC` | `#FFFFFF` 纯白 |
| **边框** | `#4A4D56` | `#FFFFFF` 纯白，3px |
| **按钮背景** | `#D4A84B` | `#000000` + 白色边框 |
| **按钮文字** | `#1A1C22` | `#FFFFFF` 纯白 |
| **正确状态** | `#4CAF50` | `#FFFFFF` + 对勾图标 |
| **错误状态** | `#F44336` | `#FFFFFF` + 叉号图标 |

**高对比度模式额外规则**：
- 所有边框宽度增加至 3px
- 所有图标增加描边
- 状态指示器必须使用图标 + 文字双重编码
- 移除所有渐变，使用纯色

### 10.4 无障碍检查清单
