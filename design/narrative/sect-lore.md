# 门派背景（派系叙事）

> **状态**: 临时
> **作者**: 用户 + Claude Code (世界构建)
> **创建时间**: 2026-03-30
> **版本**: 1.0

---

## 概述

本文档包含游戏中每个门派（派系）的叙事描述。门派代表编程语言专精，作为问题分类的背景文字。**门派没有复杂的故事线、竞争或在叙事中的活跃角色**——它们提供背景风味和玩家身份。

---

## 正典等级：临时

所有门派内容标记为 **临时**，允许在预制作期间演进。

---

## 门派叙事原则

1. **风味，非剧情**：门派背景是背景文字，不是故事内容
2. **单一身份**：每个门派有一个与现实施动语言关联的清晰身份
3. **无冲突**：门派之间不竞争或战争
4. **玩家归属**：门派选择给玩家身份感而不复杂

## MVP 门派

### Java 派（万物皆对象峰）

**英文**：Java Sect (Universal Object Peak)

#### 核心身份

| 方面 | 描述 |
|--------|-------------|
| **理念** | "一次编写，到处运行"——通过抽象实现通用 |
| **专精** | 企业系统、集合、并发、JVM 内部原理 |
| **现实映射** | Java 语言面试准备 |

#### 门派描述（玩家可见）

> Java 派追溯其传承至古老的 Sun 宗，后由 Oracle 联盟合并。世代以来，我们的弟子践行"万物皆对象修炼"的原则——从最小的整数到最大的企业系统，一切皆对象。
>
> 我们的功法强调稳定、兼容和深入理解。集合框架卷轴是我们的根基；并发心法是我们的进阶之路；JVM 内功是我们的内在修炼。掌握这些功法的修士会受到领域中最强大宗的追捧。
>
> **门派山门**：万物皆对象峰

#### 著名人物

**陈长老**（大厂 BOSS）隶属于一个重视 Java 修炼的大厂派。他严厉的方式反映了门派对深入理解而非表面知识的强调。

#### 技能领域（叙事名称）

| 领域 | 修炼名称 | 描述 |
|--------|-----------------|-------------|
| 集合 | 集合框架卷轴 | ArrayList、HashMap、HashSet 及其内部功法 |
| 并发 | 并发心法 | synchronized、volatile、ThreadPool 功法 |
| JVM | JVM 内功 | 内存模型、GC 算法、类加载 |
| Spring | Spring 剑式 | IoC 容器、AOP 原理、自动配置 |

### Go 派（云原生峰）

**英文**：Go Sect (Cloud-Native Peak)

#### 核心身份

| 方面 | 描述 |
|--------|-------------|
| **理念** | "简单即力量"——以极简达优雅 |
| **专精** | 并发系统、云原生架构、高效网络 |
| **现实映射** | Go 语言面试准备 |

#### 门派描述（玩家可见）

> 诞生于 Google 仙山，Go 派遵循"优雅简洁"的原则。我们拒绝不必要的复杂性——我们的功法以最小形式达最大效果。每行代码都是诗；每个 goroutine 都是对效率的冥想。
>
> 我们的标志性功法是 Goroutine 并发阵法——轻量级线程如水中倒影般倍增。结合 Channel 通信，我们的弟子构建可无限扩展的系统。云原生之路是我们的自然领域。
>
> **门派山门**：云原生峰

#### 著名人物

MVP 中没有 Go 派特定的 BOSS。未来 DLC 可能包含 Go 专注的面试官。

#### 技能领域（叙事名称）

| 领域 | 修炼名称 | 描述 |
|--------|-----------------|-------------|
| 基础 | 基础语法 | 变量声明、切片、映射、结构体、接口 |
| 并发 | Goroutine 阵法 | goroutine、channel、select、context、sync 包 |
| 运行时 | 运行时心法 | GC 原理、GMP 调度、内存模型 |
| Web | Web 实战功法 | gin/echo 框架、中间件、RESTful 设计 |

## BOSS 归属

| BOSS | 所属 | 备注 |
|------|-------------|------|
| **刘清**（外包HR） | 独立招聘者 | 不隶属任何门派；从所有背景招募 |
| **陈长老**（大厂技术专家） | 大厂派 | 重视深度修炼；接受所有门派展示掌握的弟子 |

### 刘清背景备注

> 刘清作为独立招聘者运作，不受限于任何单一门派。她的角色是识别有前途的修士，无论他们选择哪条道路。"好的代码就是好的代码，"她说。"我不在乎你在哪座山修炼。"

### 陈长老背景备注

> 陈长老不属于任何单一语言门派，而是代表一个重视深度而非专精的大厂派。他面试过来自每个门派的弟子，只评判修炼质量，不评判来源。
