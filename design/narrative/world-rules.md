## 概述

本文档确立修炼隐喻世界的基本规则。这些规则确保所有游戏内容的叙事一致性。**修炼世界不是真正的奇幻世界——它是应用于面试准备的隐喻视角。** 每条规则都为支持玩家的学习旅程而存在，而非构建复杂的奇幻设定。

## 正典等级：临时

本文档所有内容标记为 **临时**，允许在预制作期间演进。正式正典将在 Alpha 发布前锁定。

## 核心隐喻原则

**修炼世界是叙事构建，而非目的地。**

| 修炼元素 | 现实含义 |
|---------------------|-------------------|
| 门派 | 编程语言/框架 |
| 功法 | 知识领域（集合、并发等） |
| 修炼 | 学习和实践 |
| 境界 | 技能等级（初级 -> 高级 -> 架构师） |
| 劫难 | 面试（高压演示） |
| 心魔 | 弱点和过去的错误 |
| 飞升 | 获得 Offer |
| BOSS | 面试官 |

## 世界规则

### 1. 技术即修炼功法

**规则**：所有编程知识都在修炼隐喻中被表述为"功法"或"术"。

- 问题 = 需要掌握的修炼卷轴
- 主题 = 功法类别
- 掌握 = 深入理解一种功法
- 实现 = 练习一种功法

**示例**：
- "ArrayList 内部原理" -> "ArrayList 心经"
- "ConcurrentHashMap" -> "并发阵法"

### 2. 力量来源

**规则**：所有"力量"来自理解和练习，而非外力。

- 修炼 = 花在学习上的时间 + 正确回答的问题
- 没有魔法物品、能力提升或外部能量来源
- 所有进展通过展示知识获得

**这意味着**：
- 没有"修炼丹"或"灵石"作为物品
- 没有"幸运突破"而没有玩家行动
- 所有成长可见且由努力获得

### 3. 境界即能力标记

**规则**：境界代表现实世界的能力等级，而非任意的力量层级。

| 境界 | 现实世界对应 |
|-------|----------------------|
| 凡人 | 入门级 / 实习生 |
| 炼气期 | 初级工程师 |
| 筑基期 | 中级工程师 |
| 金丹期 | 高级工程师 |
| 元婴期 | 资深/首席工程师 |
| 化神期 | 架构师/专家 |
| 飞升 | 已获得 Offer |

**约束**：
- 境界进展是线性的（不可跳跃）
- 每个境界有具体、可展示的要求
- 境界名称使用古典修炼术语

### 4. 门派即专精

**规则**：门派代表编程语言专精，而非竞争派系。

- 门派 = 问题池和知识领域
- 没有门派竞争或冲突
- 玩家可自由切换门派
- 没有门派特定叙事或故事线

**当前门派**：
| 门派 | 现实世界 |
|------|------------|
| Java 派 | Java 语言面试准备 |
| Go 派 | Go 语言面试准备 |

### 5. 心魔即内在障碍

**规则**：心魔是过去错误的具象化，而非外部敌人。

- 它们在复习错误答案时出现
- 它们不能"攻击"玩家——它们存在是为了被承认
- 驱散心魔 = 纠正潜在的误解
- 心魔没有战斗机制

**视觉规则**：
- 心魔呈现为阴暗的剪影
- 深暗、柔和的色调
- 无恐怖或惊悚图像

### 6. BOSS 即守门人

**规则**：BOSS 是测试玩家准备程度的面试官，而非反派。

- BOSS 是中立方，非敌对方
- 他们代表真实面试场景（外包公司、大厂）
- 他们的角色是评估，而非击败
- 记忆碎片揭示他们的人性，而非恶行

**BOSS 归属**：
| BOSS | 所属 | 状态 |
|------|-------------|--------|
| 刘清（外包HR） | 独立招聘者 | 不隶属任何门派 |
| 陈长老（大厂技术专家） | 大厂派 | 资深长老 |

### 7. "威胁"

**规则**：唯一的威胁是未能充分准备。

- 没有妖魔、入侵或外部威胁
- 除了自我提升外没有时间压力
- "敌人"是玩家自己知识中的缺口
- 紧张来自挑战，而非危险

### 8. 时间与进展

**规则**：修炼世界中的时间反映现实世界的学习时间。

- 游戏世界没有"天"或"年"
- 进展以回答的问题和掌握的概念衡量
- "修炼"在游玩会话中发生
- 没有等待机制或时间门控内容

## 不存在的事物

为保持范围和隐喻一致性，以下内容 **不存在**：

| 类别 | 排除元素 |
|----------|------------------|
| **角色** | 随机 NPC、商人、任务发布者 |
| **地点** | 可探索世界地图、城镇、地牢 |
| **物品** | 装备、消耗品、收藏品 |
| **战斗** | 战斗、血量、伤害、治疗 |
| **经济** | 货币、交易、商店 |
| **社交** | 多人、公会、好友列表（MVP） |
| **政治** | 门派战争、联盟、背叛 |
| **浪漫** | 关系系统 |
| **宗教** | 神明、寺庙、崇拜 |
