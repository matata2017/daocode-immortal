> **状态**: 最终版

> **版本**: 1.0

---

## 概述

会话文本在每次游戏会话的开始和结束提供简短的氛围时刻。这些时刻故意短暂（3-5 秒阅读时间），作用为：
- 让玩家沉浸在修炼氛围中
- 为游戏玩法提供温和语境
- 在会话之间建立连贯感
- 绝不阻塞游戏玩法或感觉强制

---

## 会话开场变体

### 基于时间选择

会话开场文本应尽可能反映真实时间。

```
# 早晨（6:00 - 11:59）

"晨曦升起在修炼场上。
是时候修炼了。"

"黎明破晓。真气今晨清新。
新一天的修炼开始。"

"晨光透入。你的同道修士正在起身。
道路在等待。"

"晨雾笼罩修炼场。
专注修炼的好时机。"

"太阳刚升起。一天尚早。
今天你要掌握什么？"

# 正午（12:00 - 16:59）

"太阳高悬。修炼场安静。
专注修炼的好时候。"

"正午光芒充满练习厅。
你的修炼继续。"

"一天高峰已过。仍有成长时间。
继续你的练习。"

"下午暖意环绕你。许多修士此刻休息。
但你有事要做。"

"阳光稳定。你的真气平稳流动。
该修炼了。"

# 傍晚（17:00 - 20:59）

"傍晚将至。光线变得金黄。
即使此时，修炼也在召唤。"

"阴影在修炼场上延长。
你的专注体现在你在此。"

"一天在收尾。你的修炼不必。
继续，如果你有意志。"

"夕阳染天。练习的美丽背景。
要继续吗？"

"暮色聚集。大师说气在转换时最强。
或许他们是对的。"

# 夜间（21:00 - 5:59）

"夜幕降临。修炼场宁静。
只有专注者留下。"

"月亮升起在宁静场上。你的投入清晰。
继续你的修炼。"

"星辰满天。夜间修士知道：
黑暗带来专注。"

"大多数人已歇息。但你在这里。
道路不等人便利。"

"寂静笼罩修炼场。安静中有清明。
该修炼了。"
```

### 基于日期选择

用于能检测星期几的会话。

```
# 周一

"新的一周开始。新鲜能量。新鲜机会。
你要修炼什么？"

"周一。某事的开端。
让它有意义。"

"一周在你面前展开，充满可能。
带着目的开始。"

# 周五

"一周临近结束。但修炼没有周末。
练习继续。"

"周五傍晚。别人休息。你继续。
这把修士与凡人分开。"

"周末将至。你会休息？
还是你会成长？"

# 周末（周六/周日）

"周末。别人浪费的时间。
修士使用它。"

"今天没有义务。只有修炼。
善用它。"

"周末练习感觉不同。
完全专注成长的自由。"
```

用于连续多日回来的玩家。

```

"连续两天。重复让道路更清晰。"

# 连续第 3 天

你的修为更强了。"

继续。"

# 连续第 5 天

"五天。你在建立真实的东西。
习惯正在成为纪律。"

"一周修炼在接近。你的专注看得见。"

"第五天。练习已成你的一部分。
别停下。"

# 连续第 7 天以上

"一周或更多。你已成为真正的修士。
习惯已成身份。"

"你的连续继续。每一天加强前一天。
继续建设。"

"第 {STREAK_COUNT} 天。道路现在走熟了。
你知道方向。"

# 长连续（14+）

"两周或更多。修炼不再是练习。
它是你是谁。"

"你的连续不言自明。{STREAK_COUNT} 天。

"你已使修炼成为日常实践。
这是专注者的道路。"
```

### 中断后回来选择

用于中断后回来的玩家。

```
# 短中断（2-3 天）

"欢迎回来。修炼场想念你。
继续吗？"

"离开三天。短暂休息。
但现在，该继续了。"

"你回来了。道路在等待。
重新开始。"

# 中等中断（4-7 天）

"已过数日。你的真气可能生疏。
热身练习一下。"

"离开一周。心智遗忘缓慢。
刷新你的修炼。"

"欢迎回来。道路耐心，但别考验它。
继续你的旅程。"

# 长中断（8-30 天）

"已有一段时间。修炼场一直安静。
但它们欢迎你回来。"

"你被注意回来。距上次练习 {DAYS_AWAY} 天。
该重建势头了。"

"一个修士回来。道路仍在。
从你离开的地方开始。"

# 很长中断（30+ 天）

"自你走这条路已很久了。
但道路仍在。重新开始。"

"欢迎回来。已过许久。
但修炼耐心。重新开始。"

"已过一月多。或许新开始为宜。
道路在等待。"
```

用于任何时间，无特定语境。

## UI 展示

### 会话开场显示

```
+------------------------------------------+
|                                          |
|     [氛围视觉：修炼场                     |
|      对应时间的场景]                      |
|                                          |
|     "会话开场文本..."                     |
|                                          |
|     [点击继续]                            |
|                                          |
+------------------------------------------+
```

### 会话收尾显示

```
+------------------------------------------+
|                                          |
|     [可选：会话总结]                      |
|                                          |
|     [氛围视觉]                            |
|                                          |
|     "会话收尾文本..."                     |
|                                          |
|     [返回]                                |
|                                          |
+------------------------------------------+
```

### 显示规则

1. **可跳过**: 会话文本可立即点击跳过
2. **简短显示**: 未交互时 3 秒后自动前进
3. **可选设置**: 玩家可在设置中禁用会话文本
4. **不阻塞玩法**: 会话文本绝不阻止游戏动作

### 无障碍

- 所有会话文本可点击/轻触跳过
- 文字显示速度设置影响呈现
- 视觉背景可禁用
- 总结可单独启用/禁用

### 选择优先级

1. **基于连续**（动力最高优先）
2. **基于中断**（承认中断）
3. **基于时间**（沉浸氛围）
4. **基于日期**（每周语境）
5. **通用**（后备）

### 行长度指南

- 每行最多 80 字符
- 大多数行应为 40-60 字符
- 简短行（少于 30 字符）用于极简显示
- 统计显示用总结格式
