## 概述

本文档包含境界突破时刻的所有叙事文本。突破文本分为三个阶段：突破前提示、成功叙事、失败叙事。所有台词设计为鼓励型和成长导向，绝不惩罚。

---

## 成功叙事

### 突破仪式序列

```
阶段 1: 感觉
- 简短物理/超自然描述
- 最多 5-10 字

阶段 2: 转化
- 修士体内发生了什么变化
- 最多 10-20 字

阶段 3: 新能力
- 修士现在能做什么
- 最多 10-15 字

阶段 4: 收尾
- 氛围时刻
- 最多 5-10 字
```

### 境界特定成功叙事

```

## 失败叙事

### 设计原则

失败被描述为"尚未准备好"，绝不说"你失败了"。叙事强调道路仍然开放，玩家可以再试。

### 突破失败序列

```
阶段 1: 确认
- 发生了什么（简短，不评判）
- 5-10 字

阶段 2: 解释
- 为何不是合适时机
- 10-15 字

阶段 3: 鼓励
- 前进之路
- 10-20 字

阶段 4: 行动
- 下一步做什么
- 5-10 字
```

### 通用失败叙事

```
# 主要

"你的修为动摇。突破未成...目前。

你建立的基础还不够稳固。
这不是失败——这是获得的智慧。

继续修炼。时机成熟时，你会知道。
准备好了就回来。"

# 替代 1

"突破未达成。你的真气不稳又沉回。

不是每次尝试都成功。但每次尝试都在教导。
你的修为没有失去——它只是在等待正确的时机。

休息。练习。根基稳固后回来。"

# 替代 2

"门槛仍未跨越。暂时。

你的准备接近了，但不完整。
道路仍对你开放。它会等。

每一天修炼都让你更近。继续，回来。"

# 鼓励变体

"突破没有成形。这很常见。这很正常。

即使最伟大的修士也曾在道路上遭遇挫折。
重要的不是跌倒，而是崛起。

你的根基完好。你的真气仍在。准备好了再试。"

# 险败变体

"太接近了。你的真气差点在新境界稳定下来。

但'差点'对于永久突破还不够。
你差一步。你知道该做什么。

加强那一点弱点。然后回来。你会成功。"
```

### 境界特定失败叙事

```
# 炼气期失败

"真气难以捉摸。它不屈服于蛮力——只屈服于耐心。

"金丹开始成形，然后消散。专注不够。

金丹期需要坚定的专注和深刻理解。
你方式正确，但深度还不够。

思考你的功法。深入理解它们。然后回来。"

# 元婴期失败

"元婴无法成形。容器还没准备好。

元婴从深刻洞察中显现，不只是修为。
你的力量足够。你的理解需要深度。

寻求功法背后的原则。然后回来。"

# 化神期失败

"枷锁无法斩断。执念仍在。

化神期需要理解局限，不只是力量。
你有力量。现在寻求智慧。

思考什么阻碍了你。当你看清时，回来。"

# 飞升失败

"天界没有打开。暂时。

飞升不给未准备之人。但你很接近。
你走的每一步都导向这里。

完成你的修炼。解决剩余弱点。
当你真正准备好，天界会响应。"

## UI 展示

### 突破屏幕格式

```
+------------------------------------------+
|                                          |
|           [境界名称]                      |
|                                          |
|     [视觉效果：光芒/飞升效果]              |
|                                          |
|     "突破叙事文本..."                     |
|                                          |
|     [继续]                                |
|                                          |
+------------------------------------------+
```

### 文本显示规则

1. **逐字淡入**叙事文本（可选设置）
2. **允许跳过**在 2 秒后
3. **自动前进**在完整显示 + 3 秒后
4. **绝不阻塞**游戏玩法超过总共 30 秒

### 无障碍

- 所有突破文本可跳过
- 文字速度设置影响显示速率
- 视觉效果可禁用
- 音效伴随文本但非必需
